Project One

PourMyMindNoteApp

PourMyMind is a full-stack note-taking web application built with:

Frontend:React, hosted on Netlify at pourmymindnoteapp.netlify.app

Backend:Django REST Framework, hosted on Render at pourmymind.onrender.com

Features include:

User registration and login with username or email

JWT-based authentication via djangorestframework-simplejwt

Secure REST API endpoints, upload, update, delete, search, login, register

CORS configured to allow only the Netlify frontend

Static files served via WhiteNoise

PostgreSQL database via psycopg2

Media files handled with Pillow
